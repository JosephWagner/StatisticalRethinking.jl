rescale

Rescale a vector or an array to "un-standardize", the opposite of scale!()

Extended help

Required arguments

* `x::Vector{Float64}`                 : Vector to be rescaled
* `xbar`                               : Mean value for rescaling
* `xstd`                               : Std for rescaling

Return values

* `result::AbstractVector`            : Rescaled vector

quap

StatisticalRethinking.quap — Method

quap

Quadratic approximation of a posterior distribution

Method

quap(df)

Required arguments

* `df::DataFrame`               : Dataframe generated from a chain

Return values

* `result::NamedTuple`          : NamedTuple representing the quadratic approximation

To convert to a Dict use:

dct = Dict(pairs(result))

Example


# Run stan_sample() on a SampleModel

if success(rc)
	
	df = read_samples(sm; output_format=:dataframe)
	q = quap(df)

end
